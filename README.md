# Trader AI v2

Assistant crypto local en lecture seule.

## Ouvrir rapidement sur Windows

1. Dézippe le dossier.
2. Double-clique sur `run_windows.bat`.
3. L'application s'ouvre dans ton navigateur.

## Créer le fichier EXE

1. Dézippe le dossier.
2. Double-clique sur `build_exe.bat`.
3. L'EXE sera créé ici : `dist/Trader AI v2.exe`.

## Important

- L'application ne passe aucun ordre.
- Elle utilise les données publiques Binance Spot.
- Le score est une aide à la décision, pas une garantie.

## Fichiers

- `app.py` : interface web.
- `trader_core.py` : logique d'analyse.
- `config.json` : watchlist et réglages.
- `journal_trades.csv` : créé automatiquement pour ton journal.
- `run_windows.bat` : lancement simple.
- `build_exe.bat` : génération de l'exécutable Windows.
