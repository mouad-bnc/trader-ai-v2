# Trader AI V3

Application Streamlit simple pour suivre BTC, SOL, SUI, DOGE ou d'autres paires Binance Spot.

## Déploiement Streamlit Cloud

1. Créer un dépôt GitHub public.
2. Upload ces fichiers à la racine :
   - app.py
   - trader_core.py
   - requirements.txt
   - config.json
   - README.md
3. Sur Streamlit Cloud : New app.
4. Repository : ton dépôt.
5. Branch : main.
6. Main file path : app.py.
7. Deploy.

## Sécurité

Cette version utilise uniquement les données publiques Binance. Aucune clé API n'est nécessaire.
Elle n'exécute aucun ordre.
